# Financial Planning Advisory Website

A professional financial planning website for young professionals with integrated risk assessment and Google Sheets data collection.

## Features

- ✅ Discovery call form
- ✅ Risk assessment test
- ✅ Google Sheets integration for data collection
- ✅ Responsive design
- ✅ Professional UI/UX

## Quick Deploy to Netlify

1. **Drag & Drop Method** (Fastest):
   - Go to https://app.netlify.com/drop
   - Drag this entire folder
   - Done! You'll get a URL

2. **GitHub Method** (Recommended for updates):
   - Push to GitHub
   - Connect to Netlify
   - Auto-deploys on every push

See `DEPLOYMENT_GUIDE.md` for detailed instructions.

## Google Sheets Setup

Data is automatically saved to Google Sheets. See `GOOGLE_SHEETS_SETUP.md` for setup instructions.
